## Config и что он хранит
конфиг хранит в себе все данные, но по сути являеться json словарем
### ключи
* hash - это хеш мастер ключа, который нужен для большинства действий
* chats - список чатов, каждый чат представляет собой слвоарь со следующими ключами 
    1. indxkey - ключ шифрования для индексовы для сообщений
    2. indx - индекс чата, то есть он принимает сообщения с этим индексом,
    3. messages - текста сообщений, которые зашифрованы мастерпаролем
    4. new - булево значение есть ли новые сообщения по умолчанию false
    5. key - зашифрованный мастеключем ключ шифрования сообщений для этого конкретного чата
* init копирует данные из конфга по пути path_to_config проходиться по чатам и доабвляет нужные индексы и ключи их шифрования в список, так же индекс чата(индекс на каком месте он находитсья в self.chats)
* emails - список кортежей 0 элемент - почта, а 1 - apikey для ерверов яндекс
## функции manager
* setup - принимает аргумент password, работает если пароль не назнчен(hash_password is None) устанавлвает hash_password хешем password, используя готоовую функцию из cryptohelper(hash_password)
* create_key 
    ```python
    def create_key(self) -> str:
            str0 = "qwertyuiopaasdfghjklzxcvbnm1234567890"
            str0 = list(str0)
            return "".join([str0[random.randint(0, len(str0) - 1)] for i in range(16)])
    ```
    уже напиал эиту функцию она служит для создания ключа
* create_chat - создает пустой чат(путем добавления в self.chats), возвращает пароь для шифрованияч индексов и сам индекс
* valid_passwd - готовая функция 
    ```python
    def valid_passwd(self, password):
            return hash_password(password) == self.hash_passwd
    ```
* receive проходиться по всем email и запускает функцию receive из apihelper для каждого потом проходится по полученным сообщениям, потом вложенно проходиться по self.chats и через validate_route проверяет не письмо л это для этого чата, если это так, то дешифрует через decrypt_message и шифрует через матер кеи и полсе этого уже добавляет в messages конретного чата, так же ставт "new" в тру у чата если ему эти сообщения
